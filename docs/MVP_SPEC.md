# Artist OS MVP Spec

## Core thesis

Artists do not need another file manager. They need a song-centric workspace that keeps files, recordings, revisions, and experiments organized behind one living song.

## MVP objects

### Song
The primary creative object.

### Current Master Composition
The current source of truth for the song's structure and selected sources.

### Asset
A recording, bounce, stem, reference, or other source file. Assets are evidence, not the workspace.

### Creative Change
A structured event with:

- time
- target
- operation
- before evidence
- after evidence
- summary

## MVP screens

- Songs
- Timeline
- Assets
- DNA alpha
- Inspector

## Non-negotiable UX rule

The artist should not perform work solely to keep the software organized.

## Deferred

- Batch mix iteration
- DAW plugin integration
- Collaboration
- Public sharing
- Predictive career trajectory
